# LBTI 콘텐츠 작성 가이드

## 목적

LBTI 문항과 결과가 현실적인 로스쿨 공부 고민을 담으면서도 특정 공부법, 능력, 성적 또는 정신건강을 단정하지 않도록 하는 작성 기준이다.

## 문항 작성법

### 1. 선호 대신 충돌 상황을 묻는다

좋지 않은 예:

> 교수저를 좋아합니까?

이 문장은 현재 쓰는 교재만 측정할 수 있다. 다음처럼 두 가치가 충돌하는 순간의 첫 행동을 묻는다.

> 교수님 강의안과 암기장 문구가 다를 때 무엇을 먼저 기준으로 정리할 것인가?

응답은 `매우 그렇다`, `그렇다`, `그렇지 않다`, `전혀 그렇지 않다`의 네 단계로 고정한다. 중간값은 두지 않는다.

### 2. 한 문항은 한 축만 측정한다

“교수저를 넓게 읽고 매일 회독한다”처럼 P, W, R, S를 한 선택지에 섞지 않는다. 선택지 차이는 목표 축의 pole 하나에만 있어야 한다.

### 3. 양쪽 pole의 문장을 모두 사용한다

- 같은 축 안에서 한쪽 pole에 동의하는 문장만 반복하지 않는다.
- 한쪽만 성실하고 합리적으로 보이는 진술문을 쓰지 않는다.
- “당연히”, “무조건”, “대충”처럼 정답을 암시하는 부사를 피한다.
- 양쪽 행동 모두 현실적인 효용이 느껴지게 한다.

### 4. 동의 편향을 줄인다

축별 7개 문항에서 양쪽 pole을 지지하는 문장을 섞고 각 문항의 `agreement_pole`을 데이터에 명시한다. 사용자가 모든 문장에 습관적으로 동의해도 한 축의 한쪽 문장만 반복되지 않게 한다.

### 5. 특정 상품에 종속되지 않는다

특정 교수, 강사, 교재명 대신 교수저, 강사저, 암기장, 기출, 판례 원문처럼 일반화된 범주를 사용한다. 특정 자료가 꼭 필요한 사례라면 콘텐츠 검수 단계에서 중립성을 확인한다.

## 축별 경계

### P–T

자료의 실제 두께나 양이 아니라 충돌 시 무엇을 최종 근거로 삼는지 묻는다. P를 학문적이고 T를 요령 중심으로 묘사하지 않는다.

### W–C

책 페이지 수가 아니라 최종 학습 범위와 버리는 기준을 묻는다. C를 무조건 얇은 책, W를 무계획한 자료수집으로 축소하지 않는다.

### R–O

회독과 현출 중 부족함을 느꼈을 때 먼저 돌아가는 행동을 묻는다. 현출을 더 고급인 공부법처럼 쓰지 않는다.

### S–D

성실성과 게으름이 아니라 집중 성능이 올라가는 시간 구조를 묻는다. D를 미루는 사람, S를 완벽한 계획형 인간으로 단정하지 않는다.

## 결과문 작성법

각 유형 결과는 다음 균형을 갖는다.

1. 한 문장 정체성: 네 축의 조합이 보이는 설명
2. 대표 장면: 책상 위 또는 시험 전 행동의 구체적인 이미지
3. 강점: 이 선택 방식이 효과적인 조건
4. 주의점: 과몰입했을 때 발생할 수 있는 비용
5. 제안: 유형을 바꾸라는 처방이 아니라 반대 pole을 보완적으로 쓰는 방법
6. 유머: 사용자가 공유하고 싶지만 모욕적이지 않은 표현

“당신은 반드시”, “합격형”, “불합격형”, “머리가 좋다/나쁘다” 같은 단정은 사용하지 않는다. 결과는 현재 경향이며 시기, 과목과 환경에 따라 달라질 수 있다고 안내한다.

## 유형명과 유머

유형명은 결과 카드만 보아도 별도 해명 없이 뜻을 이해할 수 있어야 한다. 로스쿨 생활을 가볍게 과장할 수는 있지만 실제 정신건강 용어, 특정 개인·집단에 대한 비하 또는 합격 가능성을 단정하는 표현은 사용하지 않는다.

## 사용자 안내 강도

소개 또는 시작 CTA 근처에 다음 의미의 문장을 한 번만 짧게 둔다.

> LBTI는 로스쿨 공부 습관을 재미로 풀어보는 자기점검 테스트예요.

일반 질문 화면과 결과 화면에 경고문구를 반복하거나 별도 동의 모달을 띄우지 않는다. 유형별 disclaimer도 두지 않는다.

## 출처 사용 원칙

- 커뮤니티 게시물은 반복되는 논쟁의 사례로만 사용하며 로스쿨생 전체의 성향을 대표한다고 쓰지 않는다.
- 게시물 작성자의 개인 경험을 사실 검증된 일반 법칙처럼 인용하지 않는다.
- 운영 화면에 원문 문장을 길게 복제하지 않고 기획자가 재구성한 상황과 표현을 사용한다.
- 출처 목록은 `lbti-framework.json`에 유지하고 링크가 바뀌거나 삭제될 가능성을 고려한다.

## 콘텐츠 검수 체크리스트

- 문항 하나가 축 하나만 측정하는가
- 진술문이 구체적이고 한 가지 행동만 묻는가
- 양쪽 pole을 지지하는 문장이 한쪽으로 과도하게 치우치지 않았는가
- 유형별로 장점과 위험이 모두 있는가
- 특정 교재·강사·커뮤니티 이용자를 조롱하지 않는가
- 성적, 합격과 정신건강을 단정하지 않는가
- 긴 문장이 모바일에서 읽기 쉬운가
