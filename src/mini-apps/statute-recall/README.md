# 조문 리콜 (`statute-recall`)

## 목표

자주 보는 조문의 핵심 문구를 빈칸과 키워드 문제로 바꾸어 짧게 회상하는 도구입니다.

## 첫 번째 버전

- 법령명·조문 번호·조문 내용 등록
- 직접 지정한 빈칸 또는 키워드 가리기
- 입력 답안 확인과 원문 보기
- 다시 볼 조문 표시와 간단한 반복 목록
- 기기 내 자동 저장과 전체 JSON 내보내기/가져오기

## 제외 범위

- 법령 원문 자동 갱신과 외부 법령 API
- 유사 답안 AI 채점, 서버 동기화

## 데이터 방향

저장 key는 `law-solver-mini-app:statute-recall:v1`을 사용합니다. 조문 내용과 입력 답안은 GA4에 보내지 않습니다. 정답 비교 규칙은 화면과 테스트에서 명확히 고정합니다.

## 출시 조건

등록, 빈칸 생성, 답안 비교, import/export 테스트와 모바일·다크 모드 확인 후 `/apps/statute-recall`에 연결합니다.
