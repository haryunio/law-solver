# 집중 타이머 (`focus-timer`)

## 목표

공부와 휴식 구간을 반복하며 현재 학습 흐름에 집중할 수 있는 간단한 타이머입니다.

## 첫 번째 버전

- 집중·휴식 시간 설정
- 시작, 일시 정지, 재개, 초기화
- 탭이 백그라운드에 있어도 실제 경과 시간을 기준으로 보정
- 오늘 완료한 집중 구간 수의 기기 내 저장

## 제외 범위

- 서버 기반 알림, 기기 간 동기화, 상세 생산성 분석
- CBT 풀이 타이머와 상태 공유

## 데이터 방향

저장 key는 `law-solver-mini-app:focus-timer:v1`을 사용합니다. 타이머는 interval 횟수가 아니라 시작 시각과 경과 시간을 기준으로 계산합니다. 공부 시간과 완료 횟수는 GA4에 보내지 않습니다.

## 출시 조건

백그라운드 복귀, 일시 정지, 날짜 변경, 저장 복원 테스트와 모바일·다크 모드 확인 후 `/apps/focus-timer`에 연결합니다.
